# vista-agente-vscode

Statische Website zur Erklärung der neuen Agent-Ansicht von VS Code.

## Veröffentlichung mit GitHub Pages

1. Gehen Sie zu **Settings > Pages**.
2. Wählen Sie unter **Build and deployment** aus:
   - **Source:** Deploy from a branch
   - **Branch:** `main` und Ordner `/ (root)`
3. Speichern und warten Sie 1-2 Minuten.

Erwartete URL:

`https://lunasoft2001.github.io/vista-agente-vscode/`
