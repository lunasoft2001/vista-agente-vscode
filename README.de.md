## Veroeffentlichung mit GitHub Pages

1. Gehe zu **Settings > Pages**.
2. Waehle unter **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main` und Ordner `/ (root)`
3. Speichern und 1-2 Minuten warten.
