# vista-agente-vscode

Static site to explain the new Agent view of VS Code.

## Publication with GitHub Pages

1. Go to **Settings > Pages**.
2. In **Build and deployment**, select:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/ (root)` folder
3. Save and wait 1-2 minutes.

Expected URL:

`https://lunasoft2001.github.io/vista-agente-vscode/`
