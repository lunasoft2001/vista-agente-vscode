## Publication avec GitHub Pages

1. Allez dans **Settings > Pages**.
2. Dans **Build and deployment**, selectionnez:
   - **Source:** Deploy from a branch
   - **Branch:** `main` et dossier `/ (root)`
3. Enregistrez et attendez 1-2 minutes.
