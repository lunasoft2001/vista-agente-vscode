# vista-agente-vscode

Site statique pour expliquer la nouvelle vue Agent de VS Code.

## Publication avec GitHub Pages

1. Allez dans **Settings > Pages**.
2. Dans **Build and deployment**, sélectionnez :
   - **Source:** Deploy from a branch
   - **Branch:** `main` et dossier `/ (root)`
3. Enregistrez et attendez 1-2 minutes.

URL attendue :

`https://lunasoft2001.github.io/vista-agente-vscode/`
