# vista-agente-vscode

Sito statico per spiegare la nuova vista Agent di VS Code.

## Pubblicazione con GitHub Pages

1. Vai su **Settings > Pages**.
2. In **Build and deployment**, seleziona:
   - **Source:** Deploy from a branch
   - **Branch:** `main` e cartella `/ (root)`
3. Salva e attendi 1-2 minuti.
