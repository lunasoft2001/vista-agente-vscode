## Publicacion con GitHub Pages

1. Ve a **Settings > Pages**.
2. En **Build and deployment**, selecciona:
   - **Source:** Deploy from a branch
   - **Branch:** `main` y carpeta `/ (root)`
3. Guarda y espera 1-2 minutos.
