Que es la vista Agente

Es una interfaz centrada en tareas donde puedes pedir objetivos en lenguaje natural y dejar que el agente explore, edite y valide cambios de forma guiada.

Como abrirla

  1. Actualiza VS Code a una version reciente.
  2. Abre el panel de Chat/Copilot.
  3. Selecciona el modo Agente.
  4. Escribe una tarea concreta: objetivo + contexto + restriccion.

Flujo recomendado

1) Define el objetivo

"Implementa X en este repo y valida con tests existentes".

2) Revisa el plan

Confirma que cubre alcance, archivos y criterios de exito.

3) Ejecuta y verifica

Comprueba cambios, tests y comportamiento final antes de aceptar.

Buenas practicas

  • Divide tareas grandes en entregables pequenos.
  • Pide siempre validacion con build/test reales.
  • Incluye restricciones de estilo y arquitectura del proyecto.
  • Itera: feedback corto y especifico mejora mucho el resultado.
